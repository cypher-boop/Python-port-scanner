# Python Port Scanner

A simple multithreaded TCP port scanner built with Python.

## Features

* Scans common TCP ports
* Detects open ports
* Uses multithreading for faster scanning
* Works with IP addresses and domain names

## Requirements

Python 3.x

## Installation

Clone the repository:

git clone https://github.com/YOUR_USERNAME/python-port-scanner.git

Move into the project folder:

cd python-port-scanner

## Usage

Run:

python scanner.py

Enter a target IP address or domain name when prompted.

Example:

Enter target IP or domain: scanme.nmap.org

## Example Output

Scanning scanme.nmap.org...

[OPEN] Port 22
[OPEN] Port 80

Scan Complete

Open Ports:

* 22
* 80
